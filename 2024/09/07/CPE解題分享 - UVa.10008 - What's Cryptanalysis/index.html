CPE解題分享 - UVa.10008 - What's Cryptanalysis

Created2024-09-07|Updated2024-09-07
|Post Views:

題目

  • 密碼翻譯（cryptanalysis）是指把某個人寫的密文（cryptographic writing）加以分解。這個程序通常會對密文訊息做統計分析。你的任務就是寫一個程式來對密文作簡單的分析。

思路

  1. 先讀取int 到n，然後直接 while loop 做字串讀取。
  2. 每次讀取字串後，調用 freq() 統計頻率 ；直到不能再讀取。
  3. 調用 __SORT_and_Print() 做最後的排序和輸出。
    3.1 將 map 的內容轉換到 vector 這可以簡化流程；調用 std::sort() 針對 vector 進行排序 。
    3.2 排序規則依照題目所求即可。
    3.3 直接把 vector 輸出就好了。

注意事項

為了提高程式碼的可讀性，所以我選擇以封裝的形式編寫。
