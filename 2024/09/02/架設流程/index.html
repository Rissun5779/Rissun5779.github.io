Hexo 簡介

Hexo 是一個基於 Node.js 的靜態網站生成器，特別適合用來建立博客。它支援多種標記語言（如 Markdown）以及多樣化的插件，讓網站的部署和管理變得非常簡單和高效。

為什麼選擇 Hexo?

  • 簡單易用：只需要少量的指令即可完成安裝和部署。
  • 高擴展性：支援各種插件和主題，自由度高。
  • 高速：生成靜態頁面，訪問速度快，對伺服器資源的需求少。

安裝必要的工具

在安裝 Hexo 之前，請確保已經安裝了以下工具：

  1. Node.js
    Node.js 是 Hexo 的核心運行環境。請根據你的操作系統下載並安裝適合的版本。

  2. Git
    Git 用於版本控制和將網站部署到 GitHub 上。Windows 系統的用戶可以直接到官網下載安裝程式。Linux 系統用戶可以使用終端指令安裝（依照官網提供的步驟）。

  • 整個流程都在 terminal 中完成。

  • 建立hexo專案

  • 本地伺服器啟動

    • 依照 terminal 上的網址開啟網站

  • 主題和外觀配置

    • 可以到hexo官網下載和配置主題
    • 要修改預設的資訊需要進入 _config.yml 中修改

  • 將網站部署到-github

    • 登入或註冊 github 帳號

    • **點選 new repository ，並建立一個 Public 並且命名為 .github.io **

    • 安裝插件 npm install hexo-deployer-git --save

    • **到github中，申請 一個key **

    • 配置 _config.yml 文件

      在 _config.yml 文件中的 Deployment 部分，填入你的 GitHub 倉庫的 URL：

    • 到terminal中執行 hexo clean && hexo d

    • 到github-page 將source 切換為master

    • 輸入自己的網址查看是否架設成功！！

常見問題及解決方法

  1. 網站無法顯示或顯示錯誤
    • 檢查 _config.yml 中的 GitHub 倉庫 URL 是否正確。
    • 確保已正確安裝所有必要插件。
  2. 終端報錯
    • 查看報錯信息，通常是由於某些插件或依賴未安裝。
    • 嘗試運行 npm install 來重新安裝所有依賴。
  3. 參考 Hexo 的 官方文件
